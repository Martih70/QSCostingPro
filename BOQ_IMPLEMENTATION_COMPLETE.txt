================================================================================
                          QUICK START GUIDE
================================================================================

1. TEST THE MIGRATION
   npm run dev:server
   # Wait for "Database migrations completed" message

2. CREATE TEST CSV
   Item No.,Description,Unit,Quantity,Rate (£),Amount (£),Notes
   1.1.1,Test Item,item,1,100,100,Test note
   1.1.2,Test Item 2,item,1,200,200,Test note

3. PREVIEW IMPORT
   curl -X POST \
     -H "Authorization: Bearer <token>" \
     -F "file=@test.csv" \
     http://localhost:3000/api/v1/projects/1/boq-import/preview

4. PERFORM IMPORT
   curl -X POST \
     -H "Authorization: Bearer <token>" \
     -F "file=@test.csv" \
     -F "importName=Test BoQ" \
     http://localhost:3000/api/v1/projects/1/boq-import

5. EXPORT PDF
   curl -X GET \
     -H "Authorization: Bearer <token>" \
     http://localhost:3000/api/v1/projects/1/estimates/export-boq-pdf \
     -o report.pdf
